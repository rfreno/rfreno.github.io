Capstone Project: BRW Training

Application will be intuitive and user friendly. All site visitors (whether logged in or not) can view workouts that have been posted and browse a list of exercises. Exercises shown can be filtered by target muscle group, equipment used, or both. Users are guided to register or login. Once logged in, users can create workout routines that include a name, description, and list of exercises (selected from the exercises shown ). Users can choose to view details about each workout. The application will display a modal for the selected workout including tutorial images for each exercise included in the routine.

Technology Stack:

React.js, Express, Node.js, Sequelize, Postgres, bcrypt
